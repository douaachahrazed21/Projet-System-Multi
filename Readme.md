# Multimedia Systems Project

A multimedia processing and compression project developed in Python as part of the Multimedia Systems course.

## 📌 Project Overview

This project explores different multimedia processing concepts including:

- Video frame extraction
- GOP (Group of Pictures) manipulation
- Compression analysis
- Quality vs compression ratio evaluation
- Binary output generation
- Multimedia pipeline visualization

The project uses Python scripts and Jupyter Notebook implementations to experiment with multimedia encoding and compression techniques.

## ▶️ How to Run

**1. Clone the repository**
```bash
git clone https://github.com/douaachahrazed21/Projet-System-Multi.git
cd Projet-System-Multi
```
**2. Install dependencies**
```bash
pip install opencv-python numpy matplotlib
```
**3. Run scripts**

Example:
```bash
python part1.py
```
or open:
```bash
code.ipynb
```
using Jupyter Notebook.
